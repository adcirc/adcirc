Fort.11: Density Initial Condition Input File

The fort.11 file is used to specify the initial density field for baroclinic runs. Baroclinic 2DDI runs are not yet supported. Baroclinic 3D runs occur if IDEN is not equal to 0.

File Structure

The file structure varies depending on the value of IDEN in the fort.15 file.

For Baroclinic 2DDI Runs:

If IDEN = 1 or -1 (Density):

Header Line 1
Header Line 2
NVP
for k=1 to NVP
    jki, DASIGT(jki)
end k loop

If IDEN = 2 or -2 (Salinity):

Header Line 1
Header Line 2
NVP
for k=1 to NVP
    jki, DASALT(jki)
end k loop

If IDEN = 3 or -3 (Temperature):

Header Line 1
Header Line 2
NVP
for k=1 to NVP
    jki, DATEMP(jki)
end k loop

If IDEN = 4 or -4 (Temperature and Salinity):

Header Line 1
Header Line 2
NVP
for k=1 to NVP
    jki, DATEMP(jki), DASALT(jki)
end k loop

For Baroclinic 3D Runs:

If IDEN = 1 or -1 (Density):

Header Line 1
Header Line 2
NVN, NVP
for k=1 to NVP
    for j=1 to NVN
        k, j, SIGT(NHNN,NVNN)
    end j loop
end k loop

If IDEN = 2 or -2 (Salinity):

Header Line 1
Header Line 2
NVN, NVP
for k=1 to NVP
    for j=1 to NVN
        k, j, SAL(k,j)
    end j loop
end k loop

If IDEN = 3 or -3 (Temperature):

Header Line 1
Header Line 2
NVN, NVP
for k=1 to NVP
    for j=1 to NVN
        k, j, TEMP(k,j)
    end j loop
end k loop

If IDEN = 4 or -4 (Temperature and Salinity):

Header Line 1
Header Line 2
NVN, NVP
for k=1 to NVP
    for j=1 to NVN
        k, j, TEMP(k,j),SAL(k,j)
    end j loop
end k loop

Notes

  • For 3D runs, j=1 represents the bottom layer and j=NVN represents the surface layer

  • The file structure depends on the value of IDEN specified in the fort.15 file

  • Baroclinic 2DDI runs are not yet supported

  • Baroclinic 3D runs occur when IDEN is not equal to 0
