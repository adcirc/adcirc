MetGetÔÉÅ

MetGet is an application which allows users to query, format, and blend meteorological data from various sources to be used in hydrodynamic modeling applications. It serves as a meteorological forcing acquisition and development system specifically designed for hydrodynamic models like ADCIRC.

Key Features:

  • Query meteorological data from various sources

  • Format data for use in hydrodynamic models

  • Blend meteorological data from multiple sources

  • Access to storm track data (ADECK)

  • View all active storms in specified basins

  • Command-line interface for easy integration

  • Python client library for custom applications

Links:

  • GitHub Repository: https://github.com/waterinstitute/MetGet

MetGet is available under the MIT license and is developed by The Water Institute. It works in tandem with the Floodwater forecasting system.
